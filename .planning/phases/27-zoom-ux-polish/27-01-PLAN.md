**HintBarContent.swift — Create ZoomKeyCap view:**
- Private struct `ZoomKeyCap: View` taking `@ObservedObject var state: HintBarState`.
- Uses a `ZStack` with two text layers:
  - "Z" text: visible when `state.zoomFlashText == nil`, with opacity + scale transition.
  - Flash text (state.zoomFlashText): visible when non-nil, with scale+blur transition.
- Scale direction per user decision: zooming in (x2, x4) starts at scale ~0.6 and scales to 1.0; zooming out (x1) starts at scale ~1.4 and scales to 1.0. Use `state.isZoomingIn` to determine direction.
- Both transitions include `.blur(radius: 4)` during the animated phase.
- Wrap in a `KeyCap(.zoom, ...)` shell with width: 25, height: 25, same font as ESC keycap (`.system(size: 13, weight: .bold, design: .rounded)`).
- Animation: `.easeOut` with duration ~0.2s for the scale+blur. The 0.5s is the total display time before reverting, not the animation duration.
